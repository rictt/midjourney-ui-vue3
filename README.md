# Midjourney Client Web

基于[midjourney-client](https://github.com/erictik/midjourney-client)的api开发的可交互绘画界面，需要自行实现后端代理部分

## 已完成功能
- [x] 列表展示
- [x] 只看我的
- [x] 登录注册
- [x] Imagine生成图片
- [x] Upscale升级图片
- [x] 设置绘画参数
- [x] websocket接收绘画进度
- [x] pc和移动端展示

## Todo/计划
